# Lumaris v1.0.2

## 中文

### 更新
- 托盘滚轮仅在光标位于图标上时调节亮度。

### 修复
- 远程桌面等场景下，光标离开托盘后滚轮仍误调亮度。
- 误触发导致亮度浮窗自动弹出。
- 设置页滚动被当成亮度调节。

### 说明
- 覆盖安装前请退出托盘中的 Lumaris。
- 配置目录：`%LOCALAPPDATA%\Lumaris`（升级不丢设置）。

---

## English

### Highlights
- Tray scroll only adjusts brightness when the cursor is over the tray icon.

### Fixes
- Wheel still changed brightness after leaving the tray (e.g. remote desktop).
- Flyout could appear without a tray click.
- Settings page scroll was treated as brightness control.

### Notes
- Quit Lumaris from the tray before upgrading.
- Settings stay in `%LOCALAPPDATA%\Lumaris`.
